wordjumble
==========

Takes in string as input, prints out all possible words that can be formed from given word.

For example:

> python wordjumble.py dog
\ndog
>odg
>ogd
>dgo
>gdo
>god
>dg
>gd
>do
>od
>og
>go
>g
>o
>d

Handles duplicate letters:

>python wordjumble.py app
>app
>pap
>ppa
>ap
>pa
>pp
>p
>p
>a
